Tema do Projeto

Sistema LED com Arduino - Jogo de Cassino Simplificado

Resumo

Desenvolver um sistema baseado em Arduino que simule um jogo de sorte e azar. O projeto utilizará LEDs para exibir aleatoriamente três cores distintas. Se as três cores exibidas forem iguais, o jogador será declarado vencedor.

Materiais/Instrumentos

Componentes eletrônicos como Arduino, LEDs, resistores e cabos (listados na seção técnica).
